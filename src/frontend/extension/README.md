# QuranAI Browser Extension

This directory contains the source code for the QuranAI browser extension.

## Architecture

- **`index.html`**: Chat interface UI.
- **`index.js`**: Logic for managing messages and session state.
- **`manifest.json`**: Extension manifest (v3) with `host_permissions` for the local API server.
- **`bundle.js`**: Bundled JavaScript generated by Rollup (includes `shared/lib.js`).

## Build

The extension uses Rollup for bundling. To rebuild:

```bash
npm run build
```

This will output `bundle.js` which is referenced by `index.html`.

## Local API Integration

The extension connects to `http://localhost:8000/`. Ensure the QuranAI API server is running locally.
- **`createSession`**: Automatically stores `sessionId` in browser cookies.
- **`runAgent`**: Sends messages and parses the event-based array response.
