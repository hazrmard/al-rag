# QuranAI Frontend

This directory contains the frontend components for the QuranAI project.

## Structure

- **`shared/`**: Common libraries and utilities.
  - `lib.js`: JavaScript library for interacting with the QuranAI API (sessions, cookies, agent execution).
- **`extension/`**: Browser extension implementation.
  - Chat interface using the shared library.
- **`app/`**: Placeholder for future web application.

## Development

Build scripts and dependencies are managed in the root `package.json`.
